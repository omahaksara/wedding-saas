# 💌 Omah Aksara - Digital Invitation Engine

Sistem undangan digital otomatis berbasis React + Vite.

## 🚀 Cara Menambah Klien Baru
1. Buat folder baru di `public/clients/[nama-klien]/`
2. Siapkan file `data.json` dan aset foto di dalam folder tersebut.
3. Link otomatis terbentuk di: `https://omahaksara.github.io/invitation/?id=[nama-klien]`

## 🛠️ Pengembangan Lokal
1. `npm install`
2. `npm run dev`

## ⚙️ Konfigurasi Deployment
- **Base URL:** `/invitation/`
- **Host:** GitHub Pages
